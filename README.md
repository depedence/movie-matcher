# Movie Matcher

Веб-приложение для подбора фильмов в формате свайпов. Пользователь листает карточки фильмов, лайкает или пропускает — система подстраивает рекомендации под его вкус.

## Возможности

- Регистрация и авторизация через JWT
- Лента фильмов: одна карточка за раз с постером, описанием и рейтингом
- Свайп вправо (лайк) и влево (пропуск)
- Умные рекомендации на основе жанров понравившихся фильмов
- Блок последних 10 лайков на главном экране
- Данные о фильмах подтягиваются из OMDb API и кэшируются локально

## Стек

- Java 17, Spring Boot, Spring Security
- PostgreSQL, Spring Data JPA
- OMDb API
- Vanilla JS на фронте
- Selenide, Rest Assured (тесты)
